Promise allSettled
Promise allSettled는 all의 단점을 보완한 함수
all의 경우 여러 api 요청시 한가지가 에러 발생하면 나머지 다른 api 요청이 다 무산되는 경우 발생
allSettled는 오류가 발생하는 api 요청을 무시하고 나머지 api요청 정상적 작동
